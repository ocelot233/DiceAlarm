### 简介
简单的闹钟系统，会在指定窗口按照设定的时间提醒你
共有三种触发方式：`仅一次`，`每日`和`每周几`
`仅一次`闹钟会在设定的时间提醒并自动删除
`每日`闹钟会在每天的设定时间提醒
`每周几`闹钟会在每周的每个指定日指定时间提醒

模组由5个模块组成:

- 添加闹钟
- 修改闹钟
- 删除闹钟
- 查看闹钟
- 触发器

#### 使用方法：

##### 指令格式：
1. 参数间**必须**由空格分隔，连在一起的字符视为一条参数。
2. 本帖中指令示例由固定参数和指示参数构成，其中方括号为可选参数，尖括号为必选参数。使用时**请勿**包含括号

##### 指令用法：

添加闹钟：添加一个闹钟

 - 用法：`添加闹钟 <触发类型> [x天后] <x时> <x分>` **//参数间请用空格分隔**
触发类型：`仅一次`，`每日`，`每周1234567` **//(周可选数字，阿拉伯数字不分隔)。只有仅一次闹钟的x天后有效**

修改闹钟:修改已设置的闹钟

- `修改闹钟 <闹钟编号> <修改项> [对应项值]`
修改项及项值：
`更改提醒位置` **//将提醒位置更改到本窗口**
`删除提醒位置 [群号]` **//删除指定群的提醒位置，缺省参数则为删除本窗口**
`增加提醒位置` **//增加额外的提醒位置，与主窗口同时存在**
`提醒词 [提醒词]`  **//自定义提醒。不写提醒词会恢复默认**
`时间 [x天后] <x时> <x分>`
`周 12345`


我的闹钟：查看设定的闹钟。现可分页，默认显示第一页。如：`我的闹钟 1`
删除闹钟：删除指定编号的闹钟，如：`删除闹钟 1`

##### 如何启用自定义提醒词功能：
打开`alarm/script/edit_alarm.lua`，修改`enable_custom_message`值为`true`
也可以在此进行前缀权限的更改，详见注释
